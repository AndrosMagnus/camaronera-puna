# Isla Puná – 3D Terrain Map
**Camaronera Puna | Vivar Cerezo**

Interactive 3D Digital Elevation Model of Isla Puná, Guayas, Ecuador.

🗺 **[View Live Map](https://YOUR_USERNAME.github.io/camaronera-puna/)**

## Data Sources
- **DEM**: Copernicus GLO-30 (30m resolution, 2024)
- **Predio**: 188.29 ha – Juan José Vivar Cerezo
- **Zona Alta**: 183.86 ha
- **Zona de Playa**: 4.42 ha
- **Satellite imagery**: Sentinel-2 L2A (2025-09-25)

## Elevation Range
- Min: 0 m | Max: 270 m | Mean: 23 m

## Controls
- 🖱 **Drag** → Rotate terrain
- 🖱 **Right-drag** → Pan
- 🖱 **Scroll** → Zoom
- ⚙ **Vertical Scale** → Exaggerate elevation
- ⚙ **Light Angle** → Change sun direction
- ✅ **Wireframe** → Show mesh grid

## Layers visible
- Color-coded elevation (viridis scale)
- Zona Alta polygon outline (green)
- Hillshade lighting
